Before You Begin

Disable Windows Defender and any antivirus software. Overlay cheats are often flagged as false positives due to their memory access patterns. Add the loader to your exclusions list before downloading.

1

Install Required Dependencies

Download and install all required system components. These are essential for the loader to function properly.

  • DirectX 11 Runtime (Web Installer)
  • Visual C++ Redistributable 2015-2022 (x64)
  • .NET Framework 4.8 or higher
  • Windows SDK (for advanced kernel operations)
Download DirectX Runtime ‚Üí Download VC++ Redist ‚Üí Download .NET 4.8 ‚Üí
2

Disable Antivirus Protection

Critical step: Add exclusions to Windows Defender before extracting the loader.

  • Open Windows Security settings
  • Navigate to Virus & Threat Protection
  • Click "Manage Settings" under protection settings
  • Add folder exclusion for your download location
  • Disable Real-time Protection temporarily
  • Disable Cloud-delivered Protection
üîí

Security Notice

Memory-based overlays require low-level system access. This triggers heuristic detection in antivirus software. Your loader is clean but will be flagged due to its advanced hooking capabilities.

3

Extract and Run Loader

Extract VortexLoader.exe to a permanent location on your PC. Do not run from temporary folders or desktop.

  • Right-click VortexLoader.exe
  • Select "Run as Administrator"
  • If SmartScreen appears, click "More Info" then "Run Anyway"
  • Select your game from the loader interface
  • Enter your license key
‚úì

Pro Tip

Create a shortcut to the loader and set it to always run as administrator. This saves time and ensures proper permissions every launch.

4

Load Cheat & Launch Game

The loader will download and execute the overlay in the background. Once loaded, launch your game.

  • Click "LOAD CHEAT" button after entering key
  • Wait for "Loaded - Launch game now" status
  • Start your game normally
  • Overlay will automatically attach to game process
  • Default menu key is INSERT (customizable in-game)
üéØ

You're Ready

The overlay is now running. Press INSERT in-game to open the menu. All features are accessible from there. Keep the loader running in the background for optimal stability.

Troubleshooting

Loader Won't Start

  • Verify all dependencies are installed (DirectX, VC++, .NET)
  • Run as administrator
  • Check if antivirus quarantined the file
  • Re-extract from ZIP (don't run from archive)

Overlay Not Appearing

  • Ensure game is running in Fullscreen Windowed or Windowed mode
  • Press INSERT key to toggle menu visibility
  • Check if overlay process is running in Task Manager
  • Restart both loader and game

License Key Invalid

  • Verify key was copied correctly (no extra spaces)
  • Check if subscription is active
  • Ensure you're not exceeding device limit
  • Contact support for key reset if needed
